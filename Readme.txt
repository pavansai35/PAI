 Medical Cost Prediction using Linear Regression

 Project Overview

This project aims to predict medical insurance charges based on various factors such as age, sex, BMI, number of children, smoking habits, and region.

We use Linear Regression, a supervised machine learning algorithm, to build a predictive model.


Objectives

- Understand the dataset
- Perform data preprocessing
- Train a Linear Regression model
- Evaluate model performance
- Visualize predictions


 Dataset Information

The dataset contains the following features:

- age → Age of the person
- sex → Gender
- bmi → Body Mass Index
- children → Number of dependents
- smoker → Smoking status
- region → Residential area
- charges → Medical cost (Target Variable)



 Technologies Used

- Python 
- Pandas
- NumPy
- Matplotlib
- Scikit-learn



 Project Workflow

Model Evaluation Metrics
